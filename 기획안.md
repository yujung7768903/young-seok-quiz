## 퀴즈 유형
### 유형1. 너는 읽기 쉬운 마음이야~
문제 출제자 1명과 정답을 맞추는 참여자 나머지로 구성된다.
문제는 참여 인원 수 만큼 출제된다. 그리고 매 라운드마다 출제자는 모든 참여자들 중 랜덤으로 선택되며, 한 번 출제했던 사람은 랜덤 선택지에서 제외된다.
출제자의 문제는 우선순위를 선택하는 문제이며, 후보는 총 4개이다. 
선택하는 순서대로 1,2,3,4 가 표시된다.

출제자 문제 출제 화면 (제한 시간: 30초)
* 당신의 마음을 알려주세요. (1,2,3,4 순위를 모두 정해주세요.)
* 문제
* 문제 후보
* "완료" 버튼
참여자 문제 대기 화면
* "000" 님이 고민하고 있어요. 잠시 기다려주세요.
참여자가 답 맞출 때 화면(출제자 선택 완료 이후. 제한 시간: 30초)
* "000"님의 마음을 맞혀보세요.
* 문제
* 문제 후보
* "제출" 버튼
참여자 정답 대기 화면(참여자가 답변을 맞힌 후, 다른 참여자가 답변을 제출하기까지 대기)
* 제한 시간 타이머
* 제출자 닉네임 목록
완료 화면
* 맞은 비율
* 문제 후보
* 출제자 정답
* 답변자 정답
* "다음 문제로 넘어가기" 버튼 - 모든 참여자가 눌러야만 넘어감

"너는 읽기 쉬운 마음이야~" 문제 리스트
아래에서 랜덤 출제
Q. 지금 가장 가고 싶은 여행은?
1. 아기자기한 소도시 골목 투어
2. 살아숨쉬는 역사의 도시. 아름다운 건물과 미술관 투어
3. 먹는 게 삶의 낙. 맛있는 것들로 가득한 먹방 투어
4. 도파민 풀충전, 아름다운 풍경을 배경으로 액티비티 투어
Q. 애인이 동아리 또는 동호회에 든다면, 싫은 순위
1. 러닝
2. 봉사
3. 독서
4. 악기
Q. 같이 일하기 싫은 사람은?
1. 열정이 넘쳐서 실수도 넘쳐나는 후배
2. 피드백이라고 간섭을 수시로 하는 선배
3. "이게 맞아?", "확실해?"라고 계속 나한테 되묻는 상사
4. 간섭은 안하지만, 뭘 물어봐도 제대로 답변해주지 않는 회피형 선배
Q. 애인이 했을 때 가장 싫은 행동은?
1. 먹을 때 쩝쩝거리면서 다 흘리고, 같이 먹는 음식 더럽게 먹음
2. 이성 친구와 전화로 연애 고민 상담 1시간 이상
3. 가는 길에 어쩌다 마주쳐서 편의점에서 맥주
4. 옷 세일한다고 이성친구와 같은 브랜드에서 같은 디자인 옷 구매
Q. 모처럼의 긴 연휴가 왔다. 이 때 내가 하고싶은 것은?
1. 쉬는 게 짱이다
2. 시간이 아까우니 생산적인 활동을 한다.
3. 친구들과 즐겁게 논다.
4. 가족들과 시간을 보낸다.
